# 🎬 Netflix Content Analytics Dashboard

## 📌 Project Overview
This project presents an interactive **Power BI Dashboard** developed using the Netflix Titles dataset. The dashboard provides insights into Netflix's global content library by analyzing content distribution, release trends, ratings, countries, and content types.

---

## 🎯 Objectives
- Analyze Netflix content trends.
- Compare Movies and TV Shows.
- Identify top content-producing countries.
- Visualize content ratings and release patterns.
- Build an interactive dashboard for business insights.

---

## 📊 Dashboard Features
- 📌 KPI Cards (Total Titles, Movies, TV Shows, Countries)
- 🌍 Country-wise Content Distribution
- 📈 Release Year Trend
- 🎭 Content Rating Analysis
- 🎬 Movies vs TV Shows Comparison
- 🎛️ Interactive Slicers and Filters

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 Key Insights
- Movies make up the majority of Netflix's content library.
- The United States has the highest number of titles.
- Content production increased significantly after 2015.
- TV Shows represent a smaller but steadily growing share of the catalog.
